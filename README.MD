# To-Do App - JavaScript

- Descripción del Proyecto

Este proyecto es una aplicación To-Do desarrollada con HTML, CSS y JavaScript.
Permite agregar y visualizar tareas de manera sencilla desde el navegador.

Es una aplicación básica enfocada en practicar la manipulación del DOM y la lógica en JavaScript del lado del cliente.

- Este proyecto está diseñado para practicar:

Eventos en JavaScript

Uso de funciones

Manejo de arreglos

Organización básica de código

Separación de estructura, estilo y lógica

Tecnologías Utilizadas

HTML

CSS

JavaScript
